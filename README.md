# Método de Gauss

Este proyecto implementa el método de eliminación de Gauss para resolver sistemas de ecuaciones lineales.  
El código está escrito en Python y permite ingresar cualquier sistema cuadrado de ecuaciones.

## Cómo usarlo

1. Clona el repositorio.
2. Ejecuta el archivo `.py` con Python 3.
3. Ingresa el sistema de ecuaciones cuando se solicite.
